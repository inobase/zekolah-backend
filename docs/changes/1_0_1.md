> Tanggal: 2026-07-15

## 1.0.1

### Added
- **Type-check script** ‚Äî Menambahkan script `npm run type-check` menggunakan `tsc --noEmit` untuk memastikan kode bebas error TypeScript sebelum deployment
- **Banner info host & port** ‚Äî Saat dev server dijalankan (`npm run dev`), console menampilkan banner informatif berisi:
  - Host yang diaktifkan
  - Port yang digunakan
  - URL lengkap
  - Environment (development/production)
  - Versi aplikasi
  - Info API base dan health endpoint
- **Migrate script terpusat** ‚Äî Membuat `scripts/migrate.js` yang menangani semua operasi database (latest, rollback, seed) dengan otomatis memuat `.env`

### Fixed
- **Error `user already decorated`** ‚Äî Memperbaiki konflik dekorasi `request.user` yang disebabkan oleh deklarasi ganda dari `@fastify/passport` dan module augmentation. Menghapus‰æùËµñ `@fastify/passport` karena tidak digunakan (hanya JWT yang dipakai)
- **FK constraint error pada migration** ‚Äî Memperbaiki urutan pembuatan tabel di `migrations/001_create_base_tables.ts`. Tabel `students` sekarang dibuat **sebelum** `class_students` agar foreign key constraint valid (MySQL errno 150)
- **Migration script tidak load .env** ‚Äî Script migrasi sebelumnya tidak memuat `.env`, menyebabkan error "Access denied (using password: NO)". Sekarang menggunakan `require('dotenv').config()` di awal

### Changed
- **Struktur routes menggunakan decorator pattern** ‚Äî Menambahkan `app.decorate('authenticate', ...)` di `src/index.ts` sehingga semua route bisa menggunakan `app.addHook('onRequest', app.authenticate)` tanpa import manual
- **Logger sederhana** ‚Äî Mengganti logger kompleks dengan utility `src/utils/logger.ts` berbasis console yang lebih ringan untuk fase development
- **Type declaration terpusat** ‚Äî Menggabungkan type augmentation dari `types/fastify.d.ts` dan `middlewares/authGuard.ts` ke dalam `src/index.ts` menggunakan `declare module '@fastify/jwt'` dan `declare module 'fastify'`

### Files Removed
- `src/config/passport.ts` ‚Äî Tidak diperlukan, hanya JWT yang digunakan
- `src/middlewares/authGuard.ts` ‚Äî Fungsi authenticate dipindah ke decorator di `src/index.ts`
- `src/types/fastify.d.ts` ‚Äî Type augmentation digabung ke `src/index.ts`

### Files Added
- `scripts/migrate.js` ‚Äî Script terpusat untuk `latest`, `rollback`, dan `seed`
- `src/utils/logger.ts` ‚Äî Logger utility sederhana
- `scripts/` directory (baru) ‚Äî Tempat script helper CLI

### Dependencies
- **Added:** `dotenv@^17.4.2` ‚Äî Untuk memuat `.env` file di scripts
- **Removed:** `@fastify/passport` ‚Äî Tidak digunakan, hanya JWT
- **Removed:** `@types/passport-local` ‚Äî Tidak diperlukan lagi
